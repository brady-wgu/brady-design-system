# Brady Design System

Shared design assets for Brady's web and document products. Hosted via GitHub Pages.

## Aptos fonts

Self-hosted Aptos font family (Microsoft, SIL Open Font License) for use in any web project.

```html
<link rel="stylesheet" href="https://brady-wgu.github.io/brady-design-system/fonts/aptos.css">
```

Provides `'Aptos Display'` (headings), `'Aptos'` (body), and `'Aptos Mono'` (code). Matches the Aptos font used in Brady's Word/M365 documents for consistent typography across web and offline products.

Source: [Microsoft Download Center](https://www.microsoft.com/en-us/download/details.aspx?id=106087)

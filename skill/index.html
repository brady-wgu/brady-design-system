Claude Code Skill

Brady Document Style

A Claude skill that enforces Brady Redfearn's WGU document standards — applying this design system automatically to any workplace memo, brief, assessment, or report as both DOCX and Markdown.

Outputs: DOCX + Markdown WCAG 2.1/2.2 compliant Aptos typography

Overview

What the Skill Does

Invoke /brady-document-style (or just ask Claude to write a WGU document) — the skill handles the rest.

Visual System

  • ✓ Full Barn Swallow palette applied automatically
  • ✓ WGU brand blues (MIDNIGHT, NAVY, BLUE, SKY)
  • ✓ Aptos Display / Aptos / Aptos Mono typography
  • ✓ Edge-to-edge cover with optional deterministic art
  • ✓ SWALLOW accent variant for stat numbers and figures

Callout System

  • ✓ KEY INSIGHT (NAVY) — primary finding
  • ✓ WARNING (SWALLOW) — risks and caveats
  • ✓ CAUTION (BS_EMBER) — do not proceed
  • ✓ ALERT (BS_RED) — already failing
  • ✓ NOTE, BOTTOM LINE, DEFINITION, QUOTE, CODE

Writing Voice

  • ✓ No em dashes — replaced with punctuation
  • ✓ 27 banned AI-cliché phrases (auto-checked)
  • ✓ DD MMM YYYY date format enforced
  • ✓ Direct, evidence-based voice

Quality Control

  • ✓ Automated QC runs on every build
  • ✓ Auto-repairs missing keepNext on headings
  • ✓ Figure sequence, caption length validated
  • ✓ Table DXA widths, banned words, dates scanned

Usage

JSON Spec → Dual Output

Author once as a JSON spec. One command emits both DOCX and Markdown from a single walk — no divergence, no double-authoring.

Key Insight

The spec is the single source of truth. QC runs automatically on both outputs. Filename is computed from metadata — never a placeholder.

node assets/build-scripts/build-doc.js spec.json // → competency-api-caching-assessment_v1-0_10JUN2026.docx // → competency-api-caching-assessment_v1-0_10JUN2026.md { "meta": { "title": "Competency API caching assessment", "version": "1.0 (Draft)", "date": "10 Jun 2026", "profile": "assessment", "accent": "SWALLOW", "art": true }, "blocks": [ { "op": "h1", "text": "1. Problem" }, { "op": "body", "text": "At 500 sessions, latency exceeds SLA by 340ms." }, { "op": "callout", "label": "KEY INSIGHT", "body": "Cache rollout is a launch blocker.", "variant": "insight" }, { "op": "statRow", "stats": [{ "number": "340ms", "label": "SLA overage" }] } ] }

Callout System

Nine Variants

Single-cell table implementation: 4pt left accent bar + 8% tint background. Severity escalates left-to-right: NOTE → WARNING → CAUTION → ALERT.

KEY INSIGHT

Primary finding. NAVY — the strongest non-semantic signal. Bias toward this for executive findings.

NOTE

Context worth noticing. BLUE — informational, non-urgent.

BOTTOM LINE

The TL;DR recommendation. BLUE on PAPER — neutral and clean.

DEFINITION

Term introduction. BS_MID (deep steel blue) — informational.

QUOTE

Attributed excerpts. BS_TAWNY accent bar with INK body text.

WARNING

Risks and caveats. BS_YELLOW (Gape Yellow) bar; BS_EMBER label since BS_YELLOW fails as text on light backgrounds.

CAUTION

Do not proceed. BS_EMBER — warm red-orange. One level below ALERT.

ALERT

Already failing — the most severe variant. BS_RED (#BF1E2E) is a true red, clearly distinct from BS_EMBER's orange character. 6.1:1 AA.

Typography

Aptos Family

Three weights from one open-source family. Ships with Windows 11 and Microsoft 365. Self-hosted on this CDN for web use.

H1 · Aptos Display

H2 · Navy · 18pt

H3 · Graphite · 13.5pt

Body copy in INK at 11pt. Aptos for body text, Aptos Display for all headings.

Links always in BLUE #0070F0

Add to any HTML page — one line:

<link rel="stylesheet" href="https://brady-wgu.github.io/brady-design-system/fonts/aptos.css">

Then use CSS variables:

/* Headings */ font-family: 'Aptos Display', sans-serif; /* Body */ font-family: 'Aptos', sans-serif; /* Code / data */ font-family: 'Aptos Mono', monospace;

Quick Start

Use This Palette in Your Project

Copy the CSS variables block into your stylesheet — all tokens in one paste.

/* ── Brady Design System — Complete Token Set ──────────────────────── */ /* Fonts CDN */ /* @import url('https://brady-wgu.github.io/brady-design-system/fonts/aptos.css'); */ :root { /* WGU brand anchors */ --midnight: #001730; /* cover fill, H1 background */ --navy: #002855; /* H1 text, KEY INSIGHT */ --bs-mid: #1E2E56; /* DEFINITION, CODE blocks */ --blue: #0070F0; /* links, NOTE, eyebrows */ --sky: #46B1EF; /* reversed accent on dark bands */ --ice: #EEF6F9; /* light surface */ /* Brady palette */ --bs-green: #1A6635; /* ON TRACK, KPI ▲ — 7.2:1 AAA */ --bs-yellow: #FFCC00; /* WARNING, CTA on dark — 9.5:1 AAA */ --bs-tawny: #D97443; /* AT RISK, QUOTE bar */ --bs-ember: #C2592A; /* CAUTION, KPI ▼ */ --bs-red: #BF1E2E; /* ALERT, CRITICAL — 6.1:1 AA */ --bs-cream: #F7F4EF; /* warm page background */ /* Editorial neutrals */ --ink: #0A0A0A; /* body text — 19.77:1 AAA */ --graphite: #2B2F36; --slate: #4B5563; --ash: #6B7280; --fog: #9CA3AF; --mist: #D1D5DB; --paper: #F8F9FA; --bone: #F7F5EF; /* Surface tints (8% anchor into white) */ --tint-navy: #E6EBF1; /* KEY INSIGHT callout bg */ --tint-blue: #EAF3FE; /* NOTE callout bg */ --tint-green: #EDF3EF; /* ON TRACK bg */ --tint-yellow: #FFFBEB; /* WARNING callout bg */ --tint-tawny: #FCF4F0; /* AT RISK / QUOTE bg */ --tint-ember: #FAF2EE; /* CAUTION callout bg */ --tint-red: #FAEDEE; /* ALERT / CRITICAL bg */ --tint-mid: #EDEEF1; /* DEFINITION / CODE bg */ --tint-band: #DCE3EE; /* table zebra stripe */ }
